# 智慧学伴：核心知识+AI辅助

## 项目简介

"智慧学伴：核心知识+AI辅助"是智慧学伴项目展示的职业院校应用文写作教程项目展示网站。该项目通过集成七个专业AI智能体，为职业院校学生提供全方位的写作能力提升支持。

## 功能特色

### 七个专业智能体

1. **校园萌新助手** 🎒
   - 帮助新生快速适应校园生活
   - 解答入学相关问题
   - 提供学习指导和生活建议

2. **专业报告小管家** 📊
   - 协助完成各类专业报告和课程作业
   - 提供报告模板和格式规范
   - 优化内容结构和表达

3. **毕业求职助推器** 🚀
   - 提供简历优化和面试指导
   - 职业规划和求职技巧
   - 提升就业竞争力

4. **职场写作加速器** 💼
   - 提升职场写作技能
   - 优化商务文档
   - 增强职场沟通能力

### 核心功能

- **写作前准备**：学情诊断、需求分析、素材推荐
- **写作中支持**：定制模板、语言优化、结构支架
- **写作后评估**：科学测评、报告生成、个性提升

## 技术栈

- **前端框架**：Vue.js 3
- **构建工具**：Vite
- **路由管理**：Vue Router
- **样式**：CSS3 + Animate.css
- **部署**：静态网站部署

## 快速开始

### 环境要求

- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

项目将在 `http://localhost:3000` 启动

### 构建生产版本

```bash
npm run build
```

构建文件将输出到 `dist` 目录

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 智能体集成

1. 在 `src/views/Home.vue` 中配置智能体的扣子链接
2. 修改 `agents` 数组中的 `couziUrl` 字段
3. 确保链接指向正确的扣子智能体页面

### 自定义配置

- **智能体信息**：在 `Home.vue` 的 `agents` 数组中修改
- **项目信息**：在 `About.vue` 中更新项目描述
- **样式主题**：在 `src/style.css` 中修改CSS变量

## 部署指南

### 静态部署

1. 运行 `npm run build` 构建项目
2. 将 `dist` 目录上传到服务器
3. 配置Web服务器指向 `dist` 目录

### 推荐部署平台

- **Vercel**：支持Vue.js，自动部署
- **Netlify**：静态网站托管，CDN加速
- **GitHub Pages**：免费静态托管
- **阿里云OSS**：国内访问速度快
