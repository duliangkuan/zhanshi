# 智能体链接配置说明

## 📋 如何获取扣子智能体链接

### 1. 登录扣子平台
- 访问 [https://www.coze.cn](https://www.coze.cn)
- 使用您的账号登录

### 2. 找到您的智能体
- 进入"我的智能体"页面
- 找到您已经创建好的四个智能体：
  - 校园萌新助手
  - 专业报告小管家
  - 毕业求职助推器
  - 职场写作加速器

### 3. 获取智能体链接
对于每个智能体，有两种方式获取链接：

#### 方式一：从智能体详情页获取
1. 点击智能体进入详情页
2. 在页面URL中找到智能体ID
3. 链接格式：`https://www.coze.cn/store/bot/[智能体ID]`

#### 方式二：从分享功能获取
1. 点击智能体的"分享"按钮
2. 选择"复制链接"
3. 直接使用复制的链接

## 🔧 配置步骤

### 1. 打开配置文件
编辑 `src/views/Home.vue` 文件

### 2. 替换链接
找到 `agents` 数组，将以下占位符替换为实际链接：

```javascript
// 替换前
couziUrl: 'https://www.coze.cn/store/bot/您的校园萌新助手ID',

// 替换后（示例）
couziUrl: 'https://www.coze.cn/store/bot/1234567890abcdef',
```

## 🚀 测试步骤

### 1. 保存文件
保存 `src/views/Home.vue` 文件

### 2. 刷新浏览器
刷新您的开发服务器页面

### 3. 测试跳转
1. 点击任意智能体卡片
2. 观察过渡页面效果
3. 点击"立即开始对话"按钮
4. 检查是否成功跳转到扣子智能体

## ⚠️ 注意事项

### 1. 链接格式
- 确保链接格式正确：`https://www.coze.cn/store/bot/[ID]`
- 不要包含多余的空格或字符

### 2. 智能体权限
- 确保智能体设置为"公开"或"可分享"
- 检查智能体是否正常运行

### 3. 浏览器设置
- 确保浏览器允许弹窗
- 如果跳转失败，检查浏览器控制台错误信息

### 4. 网络连接
- 确保能够正常访问扣子平台
- 检查网络连接是否稳定

## 🔍 故障排除

### 问题1：点击按钮没有反应
**解决方案：**
- 检查浏览器控制台是否有JavaScript错误
- 确认链接格式是否正确

### 问题2：跳转到错误页面
**解决方案：**
- 验证智能体ID是否正确
- 检查智能体是否已发布

### 问题3：弹窗被阻止
**解决方案：**
- 允许浏览器弹窗
- 或者手动复制链接到新标签页

### 问题4：智能体无法访问
**解决方案：**
- 检查智能体权限设置
- 确认智能体状态正常

## 📞 技术支持

如果遇到问题，请检查：
1. 浏览器控制台错误信息
2. 网络连接状态
3. 智能体配置是否正确

配置完成后，您的用户就可以通过网站直接跳转到对应的智能体进行对话了！
